# 🚀 교수-학생 면담 예약 시스템 (Professor-Student Consultation System)



기존의 이메일, 오프라인 방식의 비효율적인 면담 예약을 해결하고, 교수와 학생 모두에게 편리한 일정 관리 경험을 제공하기 위한 웹 기반 예약 시스템입니다.

<br>

## ✨ 주요 기능

### 👨‍🎓 학생
- **교수님 검색**: 학과, 이름으로 원하는 교수님을 쉽게 찾아볼 수 있습니다.
- **실시간 예약**: 교수님의 면담 가능 시간을 캘린더에서 실시간으로 확인하고 예약합니다.
- **예약 관리**: 자신의 예약 현황을 한눈에 보고, 정책에 따라 예약을 취소할 수 있습니다.
- **알림**: 예약 신청, 확정, 취소 등 중요한 변경 사항에 대한 알림을 받습니다.

### 👨‍🏫 교수
- **면담 시간 관리**: 면담이 가능한 시간을 주간/월간 단위로 유연하게 설정하고 관리합니다.
- **예약 승인/거절**: 학생들의 면담 요청을 확인하고 승인하거나 거절할 수 있습니다.
- **상담 기록**: 학생과의 면담 내용을 기록하고 히스토리를 관리하여 상담의 질을 높입니다.
